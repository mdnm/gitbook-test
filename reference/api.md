# API Reference

This section provides detailed API documentation for all modules.

## Modules

| Module                                 | Description                                                                        |
| -------------------------------------- | ---------------------------------------------------------------------------------- |
| [Numerology](../modules/numerology.md) | Calculate life path numbers, cycles, challenges, and other numerological values    |
| [Gematria](../modules/gematria.md)     | Calculate Hebrew gematria values for names and words, with reverse gematria lookup |
| [Astrology](../modules/astrology.md)   | Generate birth charts and planetary positions using Swiss Ephemeris                |
| [Tarot](../modules/tarot.md)           | 78 tarot cards with meanings, spreads, and interpretations                         |
| [Ifa](../modules/ifa.md)               | Calculate Odu numbers based on dates for Ifa divination                            |
| [Core](../modules/core.md)             | Tree of Life system for building correspondences across different esoteric systems |

## TypeDoc

For complete TypeScript API documentation, generate TypeDoc:

```bash
cd kaabalah-lib
npx typedoc --out docs/api src/index.ts
```
